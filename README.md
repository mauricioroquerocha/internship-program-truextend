## Getting Started

### Prerequisites

Before running this project, make sure you have the following installed:

- Node.js (v22 or higher recommended)

- npm (comes with Node.js)

- pnpm (preferred package manager)

```bash
npm install -g pnpm
```

### Installation

```bash
pnpm install
```

## Start

```bash
pnpm start
```

Open http://localhost:3000 to see the page.
